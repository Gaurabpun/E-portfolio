This project was one of my first hands-on experiences building a website from scratch. I used HTML to create the structure of the pages, CSS to design and style the layout, and JavaScript to add interactive features. The goal was to make a simple and clean website that works well on both computers and phones. I learned how to organize content clearly and how to style it so it looks modern and easy to read.

This project helped me get more comfortable with writing code and seeing real results in the browser.

Career Connect was developed by a four-member student team as part of the Spring 2025 Hackathon at LaGuardia Community College. The platform is designed to help students especially international students find internships, mentors, and relevant campus resources more easily.

Our project includes features like major and skill-based internship filtering, links to external job platforms, and an email contact system for organizations. We focused on accessibility, clean navigation, and mobile responsiveness to ensure the tool is practical for all students. This was a collaborative experience where we divided tasks by frontend, backend, content, and UI/UX design. It deepened my understanding of teamwork in web development.
